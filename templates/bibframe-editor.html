BIBFRAME Editor

The BIBFRAME Editor is a web-based standalone editor, developed by the Library of Congress Bibliographic Framework (BIBFRAME) Initiative with with a code repository at https://github.com/lcnetdev/bfe.

The fork of the BIBFRAME Editor allows JSON-LD created in the editor to be ingested as a Fedora Object.
